# Haskell - Hello World
An example of using Haskell to print hello world in the terminal

## Install Haskell
```zsh
# ArchLinux installation
yay -S ghc --noconfirm
```

## Compile and Run the Module
```zsh
# Compile
ghc -dynamic main.hs
# Run
./main
```
